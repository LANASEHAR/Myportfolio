The problem

Many mental health apps overwhelm users with complex dashboards and excessive choices, increasing anxiety instead of reducing it.

Next

User flow

Open app → Emotional check-in → Guided action → Reinforcement → Exit

See solution

The solution

Minimal choices

Each screen shows only one main action.

Emotional safety

Rounded shapes and calm colors reduce stress.

Short microcopy

Supportive text instead of instructions.

Outcome

What this prototype shows

  • UX driven by emotional state
  • Reduced cognitive load
  • Design decisions translated into interactions
